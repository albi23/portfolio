Barcode generator is a project that was created because of the requirement to generate a significant number of bar codes and more. Most generators were limited to one time generating a single number of bar codes, so I decided to change it. This program allows you to choose one of nine types of bar codes to generate. In addition, it allows a number of modifications to the appearance of the bar code itself and preferences such as size, output format (.gif, .png, etc). The user defines whether he wants to get random data contained in the code or he defines its content. On exit, the user receives generated codes, which can also be combined into larger sheets. 
    private boolean validateChoosedImages(String input, int max, String... args) {
        StringBuilder builder = new StringBuilder();
        if (args != null)
            for (String s : args) builder.append(s);

        builder.insert(0, input);
        String[] inputData = fillWithComma(builder.toString()).split(",");
        validated = new LinkedHashSet<>();

        for (String s : inputData) {
            if (s.contains("-")) {
                String[] res = s.split("-");
                for (int i = Integer.parseInt(res[0]); i <= Integer.parseInt(res[1]); i++) {
                    if (i > max) return false;
                    else validated.add(i);
                }
            } else {
                validated.add(Integer.parseInt(s));
            }
        }
        return true;
    }
