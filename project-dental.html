Dental Clinic Database

Description

Code example

Dental Clinic Database was a project implemented during the third semester of study. It presents a fully functional panel for many levels of clinic service. The main task is to collect information on appointments so that the doctor can control his time in a clear and transparent way. The application has 3 levels of access for: Receptionist, Doctor and Facility Director. Each of them has a convenient panel in which it can enter new data. 
    public static ForceGenerateGui createGui(LogInPanel logInPanel, String type, Stage mainWindow){
        switch (type){
            case "Administrator":
                return new AdministratorGui(logInPanel, mainWindow);
            case "Lekarz":
                return new DoctorGui(logInPanel,mainWindow);
            case "Recepcjonistka":
                return new ReceptionistGui(logInPanel,mainWindow);
            case "Dyrektor":
               return new DirectorGui(logInPanel, mainWindow);
        }

        return null;
    }
