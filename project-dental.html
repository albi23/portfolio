Bank Controller

Description

Code example

Bank Controller is an application that uses many technologies, including Angular, Docker, MySql, and SpringBoot. The main task of the application is to collect information on individual loans, as well as display information on borrowers. The main basis of the application is Angular, from where we have the entire control room by individual steps. The data entered by the user are sent to a server built on Spring and then saved in the MySQl database. 
    @CrossOrigin
    @RequestMapping(
            method = RequestMethod.POST,
            path = "/Products",
            produces = MediaType.APPLICATION_JSON_VALUE
    )
    public @ResponseBody
    List<Product> getProducts(HttpServletRequest request) {

        List<Product> productsInfo = new ArrayList<>();
        try {
            List<String> creditsIdObject = IOUtils.readLines(request.getInputStream(), "UTF-8");
            String result = creditsIdObject.get(0).substring(1, creditsIdObject.get(0).length() - 1);
            String[] idCredits = result.split(",");
            for (String id: idCredits) {
               this.productRepository
               		.findById((Integer.parseInt(id)))
               		.ifPresent(productsInfo::add);
            }
        } catch (Exception ex) {
            ex.printStackTrace();
        }
        return productsInfo;
    }
